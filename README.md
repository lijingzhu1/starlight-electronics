# Starlight Electronics LLC Website Prototype

This is a low-cost static website prototype for Starlight Electronics LLC.

Recommended deployment:
- GitHub Pages for free hosting.
- Optional custom domain later.
- Static HTML/CSS/JS so the site is simple, fast, and easy to edit.

Initial pages:
- Home
- Graphic Overlays
- Membrane Switches
- Industries
- Camera Lens Protective Films
- Request a Quote
- Patents & Documentation
- Contact Us

Launch files:
- `marketing-launch-plan.md` for the first low-cost outreach and ads test.

Before launch:
- RFQ/contact email is currently set to `starlight.rfq@outlook.com`.
- If using Google Ads, send traffic to product pages or `request-quote.html` with UTM parameters.
